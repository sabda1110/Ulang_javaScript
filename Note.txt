-console
javascript memiliki fitur untuk melakukkan logging bernama console
logging adalah mekanisme yang bisa dilakukan oleh programmer untuk menampilkan informasi dari aplikasi yang sedang berjalan,tanpa harus menggangu alur kerja aplikasi dan juga interaksi user
untuk melakukan ini,kita bisa menggunakan fitur console di javaScript
untuk menggunakan console,kita cukup gunakan perinctah console di javaScript

Console Method       -      Keterangan
console.info(...)           Memberikan informasi
console.warn(...)           Memberitahu informasi peringatan
console.error(...)          Memberitahu informasi error
console.table(...)          Memberitahu informasi dalam bentuk table
